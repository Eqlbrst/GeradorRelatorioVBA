# GeradorRelatorioVBA
Ferramenta criada com base na ABS Consulting para a Análise de Causas. A funcionalidade é simples, nos campos existentes dentro da ferramenta, o usuário
coloca as respostas e a caracterização das causas, o programa irá realocalas para dentro de um documento no word, que será criado no momento que o usuário finalizar o preenchimento.
Claro que dentro disso temos algoritimos complexos para geração de documento, uma vez que o layout da página não é tao simples.

Caso queira baixar o arquivo para teste, é importante deixar claro que a ferramenta é feita em VBA e ultiliza macros.
